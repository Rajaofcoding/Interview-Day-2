Question: Explain br tags 

 Ans: The br tag is a self-closing element, meaning it doesn’t require a closing tag.
      web browsers don’t automatically recognize new lines or paragraph formatting. This is where the br tag comes into play.

Question: Explain how many different types of lists we have in html 

 There are 2 types of lists in html that we use to list items
        These are the following lists
        1. Ordered list ( An ordered list starts with the  < ol >   tag. Each list item starts with the  < li >  tag.)
        2. Unordered list ( An unordered list starts with the  < ul >   tag. Each list item starts with the  < li >  tag.)


Question: What are semantic tags 

 Ans: Semantic HTML tags are tags that define the meaning of the content they contain. 
      For example, tags like header tag, article tag, and footer tag are semantic HTML tags. They clearly indicate the role of the content they contain.
